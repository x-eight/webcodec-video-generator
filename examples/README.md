# Examples

This directory contains example scripts demonstrating how to use the WebCodec Video Generator.

## Available Examples

### Basic Example (`basic.ts`)
A simple example showing:
- Single image clip
- Single text clip
- Basic timing configuration
- Minimal setup

**Run:**
```bash
npm run example:basic
```

### Advanced Example (`advanced.ts`)
A comprehensive example demonstrating:
- Multiple overlapping clips
- Complex timing sequences
- Custom video options
- Layer ordering (zIndex)
- Opacity and rotation effects

**Run:**
```bash
npm run example:advanced
```

## Output

All examples generate video files in the `output/` directory at the project root.

## Customization

Feel free to modify these examples to experiment with:
- Different image URLs
- Custom text content
- Various timing configurations
- Different resolutions and frame rates
- Multiple layer compositions
